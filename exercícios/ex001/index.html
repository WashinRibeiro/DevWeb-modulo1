HTML (Hypertext Markup Language). Ou seja, Linguagem de Marcação de HiperTexto, é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web.
